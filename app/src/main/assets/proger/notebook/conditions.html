Операторы сравнения
> - больше
>= - больше или равно
< - меньше
<= - меньше или равно
== - равно
!= - не равно

Операторы можно использовать для двухстороннего сравнения
if 1 < a < 10: # если a больше 1 и меньше 10
 print("1 < a < 10")

Условия
if 1 != 3: # if - обязательная часть условия
 print("1 is not 3")
elif 1 == 2: # elif - можно не использовать
 print("1 is 2")
else: # else - можно не использовать
 print("1 is 3 and not 2")

Вложенные условия
a = int(input())
if 4 <= a <= 16:
 if 9 <= a:
  print("9 <= a <= 16")
