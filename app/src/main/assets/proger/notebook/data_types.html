int : тип данных, обозначающий целые числа (4, -3, 0)
float : тип данных, означающий числа с плавающей точкой (74.3, 345.855345)
string : тип данных, означающий строки ('dunz', "mobster"). Тип, в котором Python принимает данные
boolean : тип данных, означающий логические переменные (может принимать только значения True или False)
